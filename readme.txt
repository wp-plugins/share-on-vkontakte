== Description ==

Add link to share the current post or page to social network VKontakte (vkontakte.ru, vk.com). Just setup in 1 click.

== Installation ==

1. Upload folder `share-on-vkontakte` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings/Share on Vkontakte to configure the button
4. Enjoy!
